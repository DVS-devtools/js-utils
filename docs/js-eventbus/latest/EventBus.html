    EventBus

    new EventBus()

    Simple library to trigger custom events

    Source:
    • index.js, line 14
    Example
    import EventBus from '@docomodigital/js-eventbus';

const Bus = new EventBus();
const callback = (args) => {
  // Do something...
};

Bus.on('customEvent', callback);
Bus.trigger('customEvent', {foo: 'bar'}); // callback is called

    on(eventType, func, contextopt) → {void}

    Register a new callback for the specified custom event

    Source:
    • index.js, line 40
    Parameters:
    Name Type Attributes Default Description
    eventType String

    if not exists it defines a new one
    func function

    the function to call when the event is triggered
    context Object <optional>
    		 null

    the 'this' applied to the function. default null, this will not work with arrow functions.
    Returns:
    Type:
    void
    Example
    const Bus = new EventBus();
Bus.on('customEvent', (args) => {
    // Do something...
});

const ctx = {foo: 'bar'}
Bus.on('customEvent', (args) => {
    console.log(this.foo) // Uncaught TypeError: Cannot read property 'foo' of undefined
}, ctx);
Bus.on('customEvent', function(args) {
    console.log(this.foo) // 'bar'
}, ctx);
