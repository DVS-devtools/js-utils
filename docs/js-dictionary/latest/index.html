    Library to get Dictionary values

    Usage

    import Dictionary from '@docomodigital/js-dictionary';

const dictObject = {
    SEARCH: 'Search!',
    GO_BACK: 'Go Back',
};

Dictionary.init({
    dict: dictObject,
    showKey: 'missing'
});

Dictionary.get('SEARCH'); // 'Search!'

    Installation

    NPM

    npm install --save @docomodigital/js-dictionary
