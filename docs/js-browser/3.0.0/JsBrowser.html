    JsBrowser

    new JsBrowser()

    JsBrowser provides you an alternative method to manage previous and current page changements. Is particularly used to get query params as object and to get the previous page in order to implement an easy and basic history in your routing

    (static) getPrevPage() → {String}

    Return the previous page you visited, if is empty, then returns null.

    Source:
    • Browser/index.js, line 50
    Returns:
    Type:
    String

    previous page visited (ie: '/home')

    Example

const prevPage = Browser.getPrevPage(); // '/home'

    (static) shiftPage(currentPage)

    Adapt the Browser.prevPage and Browser.currentPage according to route's changement.

    NB: route changements must be specified in your routing changement function observer

    Source:
    • Browser/index.js, line 25
    Parameters:
    Name Type Description
    currentPage String

    set the previous page to the actual Browser.currentPage, then set the Browser.currentPage to the given new value
